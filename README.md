# picam

Libcamera capture PoC

*Note:* Requires [libcamera](https://libcamera.org/) development packages. You have two options:

- Install using package manager: `apt install rpicam-apps libcamera-dev` 
- Build raspberry-pi fork of libcamera: [raspberrypi/libcamera](https://github.com/raspberrypi/libcamera). 

(Libcamera development is very active. With the latter option, you get the latest version)
